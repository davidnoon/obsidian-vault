# Agent Memory Vault

This vault is synced across all devices using Obsidian Sync.

## Structure

- **00-Inbox** - Quick capture and notes to process
- **01-Projects** - Active and archived projects
- **02-Contacts** - People database
- **03-AI-Sessions** - Chat exports from Claude, Gemini, OpenClaw, Antigravity, OpenCode
- **04-Policies-Rules** - Agent prompts, system prompts, and configurations
- **05-Migrated** - Imported Apple Notes, Evernote, Word, PowerPoint, Spreadsheets
- **06-Media** - Images and attachments
- **07-References** - Research and external links

## Plugins

- **Local REST API** - Port 27123 for agent access
- **Dataview** - Database queries
- **QuickAdd** - Fast capture
- **Todoist** - Task management

## Agent Access

Agents can access this vault via:
- Tailscale network to this desktop
- Local REST API at http://[this-ip]:27123

## Sync

Sync enabled via Obsidian Sync - sync status shown in bottom-left of window.
